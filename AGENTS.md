# Agent Instructions

This repository is compliance-first.

## Product frame

Real Match is a private verified social-introduction and cultural concierge platform for consenting adults.

## Preserve these boundaries

- Do not describe the service as matchmaking, a marriage agency, or a mail-order bride service.
- Do not promise matches, meetings, relationships, chemistry, marriage, sex, companionship, or any romantic outcome.
- Do not frame women or any member as inventory, supply, product, access, or reward.
- Do not call the concierge coordination fee a success fee.
- Do not add backend, auth, database, payments, ID uploads, or member messaging unless specifically requested.
- Prefer boring compliance, auditability, consent, data minimization, and safety over flashy features.

## Engineering rules

- Keep App Router routes simple and static until backend scope is approved.
- Run `npm run lint`, `npm run typecheck`, and `npm run build` after code changes when available.
- Document assumptions in the changed docs or PR summary.
- Do not commit secrets.
- Keep `.env.example` safe and placeholder-only.

## Copy guardrail

Approved language: verified social introductions, private assessment, profile consultation, member standards, optional logistical concierge coordination.

Forbidden language: matchmaking, marriage broker, Filipina wife, handpicked girls, guaranteed match, success fee, access to women, sexual access, buy introductions.
